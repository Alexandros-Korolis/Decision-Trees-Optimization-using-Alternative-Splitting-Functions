# Decision Trees Optimization using Alternative Splitting Functions

## Folder Contents

• The IRIS, WINE, and HABERMAN directories each include an .R file containing the corresponding analysis, along with an .xlsx file that stores the results.

• The SOFTX-D-20-00097 directory contains a modified version of the original code, extended to include the Kaniadakis entropy. The original package is available at:
https://github.com/ElsevierSoftwareX/SOFTX-D-20-00097

• The Helper Functions directory contains an .R file that includes functions for performing grid search.
